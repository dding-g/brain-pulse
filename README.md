## 1. Executive Summary

### 한 줄 요약

> **"비싼 구독과 지루한 훈련에 지친 유저들을 위한, 매일 5분 뇌 컨디션 체크 앱"**

### 핵심 가설

| 항목 | 내용 |
|------|------|
| **문제** | 기존 뇌 훈련 앱(Lumosity, Peak)은 비싸고($35~80/년), 지루하고, "훈련 의무감"으로 인해 이탈률이 높다 |
| **솔루션** | "훈련"이 아닌 **"측정"** 프레이밍. 매일 뇌 컨디션을 체크하고 추적하는 습관형 앱 |
| **차별화** | 무료(광고 기반) + AI 적응형 난이도 + 공유 가능한 결과 카드 + 로컬 퍼스트 프라이버시 |
| **수익 모델** | 보상형 영상 광고 중심 → DAU 10K+ 시점에 하이브리드(구독 추가) |
| **타겟** | 한국 20~30대 직장인/학생, 이후 글로벌 확장 |

### 핵심 전략 비유

```
기존 앱 = "뇌 헬스장"   → 가야 한다는 의무감 → 이탈
BrainPulse = "뇌 체중계" → 올라가보는 호기심 → 습관화
```

사람들은 운동(훈련)은 귀찮아해도, 자신의 상태(데이터)를 확인하는 것에는 중독성을 느낀다. 이 심리적 차이가 리텐션의 핵심.

## 3. 제품 전략

### 3.1 포지셔닝

```
기존:  "과학적 뇌 훈련" → 비싼 구독 → 의무감 → 이탈
우리:  "매일 5분, 뇌 컨디션 체크" → 무료(광고) → 재미 → 습관화
```

| 축 | 기존 앱들 | BrainPulse |
|----|----------|-----------|
| 프레이밍 | "훈련하세요" | "오늘 뇌 온도가 몇 도인지 체크해보세요" |
| 가격 | $35~80/년 | 무료 (광고 기반) |
| 동기 부여 | 의무감 + 죄책감 | 호기심 + 자기 데이터 추적 |
| 공유 | 없음 | SNS 공유 카드 (바이럴 엔진) |
| 데이터 | 클라우드 필수 | 로컬 퍼스트 (프라이버시) |
| AI | 고정 난이도 테이블 | 실시간 적응형 난이도 |

### 3.2 타겟 유저 페르소나

**Primary: 한국 20~30대 직장인/학생**

| 속성 | 설명 |

### 4.2 사전 컨디션 체크 시스템

앱 시작 시 5초 내로 3가지 컨디션 요소를 수집. 이 데이터는 두 가지 역할:

1. **AI 모드 추천**: "수면 5시간 + 기분 낮음 → 휴식 모드 추천"
2. **리포트 신뢰도 강화**: 게임 결과 + 컨디션 데이터 결합 → 상관관계 인사이트

```
수면 5시간 + 카페인 0잔 → 집중력 -12%
  → "수면 부족이 영향을 준 것 같아요"
  → 유저 반응: "아 점수가 낮은 게 내 잘못이 아니라 수면 탓이구나" (납득)
  → 앱 신뢰도 ↑, 다음 날도 측정하고 싶은 동기 ↑
```

### 4.3 세 가지 모드 상세

#### 🧘 휴식 모드 — "뇌 쿨다운"

**사용 시점**: 아침 기상 직후, 점심 후, 잠들기 전  
**목표**: 스트레스 해소, 마음 정리  
**뇌과학 원리**: 반복적이고 예측 가능한 행동이 코르티솔을 낮추고 알파파(Alpha wave) 활성화

| 미니게임 | 메커니즘 | 시간 | 구현 난이도 |
|---------|---------|------|-----------|
| **컬러 매칭** | 같은 색 타일 터치 (타이머 없음) | 60초 | ★★☆ |
| **호흡 버블** | 호흡 가이드에 맞춰 버블 크기 조절 | 60초 | ★★☆ |
| **패턴 완성** | 느린 속도로 패턴 이어그리기 | 60초 | ★★☆ |

**측정 지표**: 터치 안정성, 반응 일관성 → **"안정 지수"** 산출

#### ⚡ 활성화 모드 — "뇌 워밍업"

**사용 시점**: 업무/공부 시작 전, 회의 전  
**목표**: 집중력과 처리 속도 끌어올리기  
**뇌과학 원리**: 베타파 활성화, 도파민 분비, 전두엽 & 두정엽 활성화

| 미니게임 | 메커니즘 | 시간 | 구현 난이도 |
|---------|---------|------|-----------|
| **스피드 매칭** | 이전 것과 같은지/다른지 판단 | 45초 | ★☆☆ |
| **컬러 스트룹** | 글자색과 의미 불일치 판별 | 45초 | ★☆☆ |
| **순간 기억** | 숫자/도형 순서 기억 후 재현 | 45초 | ★☆☆ |
| **빠른 산술** | 간단한 사칙연산 빠르게 | 45초 | ★☆☆ |

**측정 지표**: 정확도 × 속도 → **"집중 지수"** 산출

#### 🧠 발달 모드 — "뇌 트레이닝"

**사용 시점**: 여유 있는 시간, 주말  
**목표**: 논리력, 공간 지각, 전략적 사고 강화  
**뇌과학 원리**: 전두엽 실행 기능 강화, 신경 가소성 촉진

| 미니게임 | 메커니즘 | 시간 | 구현 난이도 |
|---------|---------|------|-----------|
| **미니 스도쿠** | 4×4 → 6×6 → 9×9 적응형 | 2~3분 | ★★☆ |
| **탱그램** | 도형 조합으로 실루엣 완성 | 1~2분 | ★★★ |
| **경로 탐색** | 최단/최적 경로 찾기 | 1~2분 | ★★★ |
| **N-back** | N번 전 자극 기억 비교 | 1~2분 | ★★☆ |

**측정 지표**: 문제 난이도 × 성공률 → **"성장 지수"** 산출

### 4.4 콘텐츠 고갈 방지 시스템

게임 수를 늘리는 것이 아니라, **같은 게임의 변주를 무한하게** 만드는 구조:

| 레이어 | 전략 | 구현 방식 |
|--------|------|----------|
| **문제 세트 무한 생성** | 로직 고정, 파라미터 랜덤 | 스도쿠 생성 알고리즘, 산술 숫자 랜덤, N-back 자극 랜덤 |
| **시각 테마 로테이션** | 월별/시즌별 테마 교체 | 색상 팔레트, 배경, 사운드만 변경 (게임 로직 불변) |
| **데일리 모디파이어** | 매일 변형 규칙 적용 | "오늘은 역순으로!", "오늘은 시간 절반!" 같은 모디파이어 |

```
콘텐츠 = 게임 로직(고정) × 파라미터(무한) × 테마(시즌) × 모디파이어(데일리)
```

Wordle이 매일 단어 하나만 바꿔도 1년 넘게 유지되는 것과 같은 원리.

## 5. AI 적응형 시스템

### 5.1 왜 AI 적응형이 킬러 피쳐인가

**기존 앱의 문제**:
```
Level 1 → Level 2 → Level 3 → ... → Level 50
(고정 난이도 테이블, 모든 유저 동일 경험)
```

- 너무 쉬우면 지루함 → 이탈
- 너무 어려우면 좌절감 → 이탈
- "플로우 상태"에 도달하는 유저가 극소수

**BrainPulse의 접근**:
```
                 ┌── 불안 (너무 어려움)
                 │
    난  ─────────┤    🎯 플로우 존
    이           │   ↗ (AI가 유저를 여기에 유지)
    도  ─────────┤  ↗
                 │ ↗
                 ├── 지루함 (너무 쉬움)
                 └──────────────────────→
                        유저 스킬 레벨
```

### 5.2 구현 단계

#### Phase 1 — 규칙 기반 (MVP)

```javascript
function adjustDifficulty(history) {
  const recentResults = history.slice(-5);
  const accuracy = recentResults.filter(r => r.correct).length / 5;
  const avgTime = recentResults.reduce((sum, r) => sum + r.time, 0) / 5;

  if (accuracy > 0.85 && avgTime < targetTime * 0.8) {
    return currentLevel + 1;  // 너무 쉬움 → 올림
  } else if (accuracy < 0.5 || avgTime > targetTime * 1.5) {
    return currentLevel - 1;  // 너무 어려움 → 내림
  }
  return currentLevel;        // 적절 → 유지
}
```

#### Phase 2 — ML 기반 (성장 후)

```
유저 프로필 벡터 = [기억력, 논리력, 속도, 공간지각, 주의력]
                    ↓
          다차원 능력 모델 (IRT: Item Response Theory)
                    ↓
          개인화된 게임 세트 + 난이도 추천
```

### 8.2 백엔드 기술 비교 (Supabase vs Cloudflare Workers)

기존 Supabase Edge Functions 대신 **Cloudflare Workers + Hono** 채택 근거:

| 비교 항목 | Supabase Edge Functions | Cloudflare Workers (선택) |
|-----------|------------------------|--------------------------|
| **무료 요청** | 500K/월 | 100K/일 (~3M/월) — **6배** |
| **Cold Start** | ~42ms | ~0ms (V8 Isolates) |
| **무료 DB** | Postgres 500MB | D1 5GB + 5M reads/일 |
| **이미지 생성** | 제한적 (Deno 런타임) | workers-og (Satori 기반) |
| **AI 추론** | 외부 API 호출 필요 | Workers AI 100K req/일 무료 |
| **월 비용 (MVP)** | $0 | $0 |
| **월 비용 (DAU 50K)** | $25/mo (Pro 필수) | $5/mo |

**결론**: MVP부터 DAU 50K까지 **월 $0~5**로 운영 가능. Supabase 대비 6배 많은 무료 요청, 제로 콜드 스타트, 내장 AI 추론까지 제공.

### 8.3 기술 결정 근거

| 결정 | 이유 |
|------|------|
| **Expo SDK 52+** | EAS Build로 광고 SDK 통합, OTA 업데이트, prebuild 지원 |
| **react-native-skia** | 미니게임 렌더링에 Canvas 수준 성능 필요, RN에서 가장 빠름 |

## 12. 부록: 퍼즐 게임 분류 레퍼런스

### 12.1 뇌 기능별 게임 분류

#### 🧘 뇌 휴식 게임 (Brain Rest)

> 특징: 타이머 없음, 실패 페널티 없음, 반복적 패턴, ASMR적 만족감

**매칭 & 정렬 계열**: Match-3 (Candy Crush, Royal Match), 타일 매칭 (Mahjong Solitaire), 컬러 소트 (Water Sort), 블록 정렬 (Block Blast, 1010!), 버블 팝 (Bubble Shooter)

**조립 & 완성 계열**: 직소 퍼즐, 넌오그램/피크로스, 컬러링, 도트 연결 (Two Dots), 실 풀기 (Yarn Untangle)

**환경/시뮬레이션 계열**: Mini Motorways, Mini Metro, 정원 가꾸기 (Prune), 모래/물 시뮬레이션

#### ⚡ 뇌 활성화 게임 (Brain Activation)

> 특징: 시간 제한, 빠른 판단, 즉각적 피드백

**반응속도 & 주의력**: 속도 매칭 (Lumosity), 스트룹 테스트, N-back, 빠른 산술 (Elevate), 반응 게임 (Piano Tiles)

**패턴 인식 & 시각 탐색**: 틀린 그림 찾기, 숨은 물체 찾기, 카드 뒤집기 (Memory Match), Simon Says, Wordle 계열

**순간 전략**: 퀴즈/트리비아 (Trivia Crack), 단어 만들기 (Boggle, Scrabble GO), 카드 속도전 (Solitaire)

#### 🧠 뇌 발달 게임 (Brain Development)

> 특징: 깊은 사고, 장기 전략, 다단계 계획

**논리 추론**: 스도쿠, KenKen, 마인스위퍼, 논리 그리드 퍼즐

**전략 & 계획**: 체스, 바둑, 오셀로, 장기

**공간 지각**: 탱그램, 루빅스 큐브, Monument Valley, The Room 시리즈, 소코반

**멀티 인지 훈련**: Lumosity, Peak, Elevate, NYT Crossword, 2048/Threes!, 탈출 게임

**전략 보드게임 (모바일 앱 존재)**: 카탄 (Catan), 스플렌더 (Splendor), 아줄 (Azul), 티켓 투 라이드, 팬데믹, 7 Wonders
